Elliot Jimenez

One the first delivery I was do the windows system, "implementing" Docking system.

For the second delivery, I did the entire Resources system. In our engine we have 3 types of Resources:

  • Resource Mesh
  • Resource Material.
  • Resource Script.

All Resources are imported, undownloaded from memory and reloaded into memory, all this automatically.

For the last installment, implement a scripting system, I worked side by side with my coworker. It has been one of the most complex systems to implement.

For a future I would like to clean the code, restructure some systems that are not very well optimized and finish the engine with the other systems: audio, physics, particles, UI,
